# Technical Documentation Page - C# Documentation

Este repositório contém um projeto de **Página de documentação técnica** desenvolvido como parte dos desafios do FreeCodeCamp.

## 📌 Descrição
A **Página de documentação técnica** documenta a **linguagem de programação C#**, mostrando os tipos de dados, declaração de variáveis e estruturas de controle de fluxo.

## 🔧 Tecnologias
- **HTML**: Estrutura da página
- **CSS**: Estilização e layout responsivo

## 🎯 Funcionalidades
- ✅ Navegação entre os tópicos da documentação
- ✅ Demostração de blocos de código
- ✅ Responsividade
