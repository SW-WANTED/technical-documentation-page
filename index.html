Introdução

A linguagem C# é a linguagem mais popular para a plataforma .NET, um ambiente de desenvolvimento gratuito, multiplataforma e de código aberto. Os programas C# podem ser executados em muitos dispositivos diferentes, desde dispositivos de Internet das Coisas (IoT) até a nuvem e todos os outros lugares. Você pode escrever aplicativos para telefones, desktops e laptops e servidores.

Variáveis

As variáveis em c# são fortemente tipadas, o que significa que você terá de definir os tipos de dados na sua declaração

Na declaração de variáveis você pode:

  • Definir o seu tipo de dados
  • Inicializar a variável com um valor
  • Definir um tipo var para o compilar decidir qual tipo de dados escolher com base no valor atribuido.

Declaração de uma variável:

C# bool isAlive;

Inicializando uma variável:

C# string nome = "SW Wanted";

Definindo um tipo var para a variável:

C# var idade = 20;
Estruturas de Controlo

As estruturas de controlo são comandos que nos permitem alterar o fluxo de execução normal do programa para um outro quando uma condição lógica for satisfeita.

Por exemplo, tu foste contratado para fazer um aplicativo que permita realizar transferências bancárias. Para isso, tu crias elementos que permitam os usuários inserir os dados necessários á transferência: 'IBAN do Destinatário' e o 'Montante', e de seguida, tu crias um botão que permite realizar a transferência bancárias de acordo com os dados fornecidos. O normal a acontecer é que após o botão ser clicado, a transferência deve ser feita. O problema é que nem sempre terás o saldo suficiente para fazer esta operação. Daí surge a necessidade de alterar este 'Fluxo normal' para um outro, que é exibir uma mensagem de erro quando o saldo não for suficiente.

As estruturas de controlo em C# são:

  • A instrução if...else
  • A instrução switch...case
C# // Esta é uma sintaxe simples da instrução if if (condition) { /* Seu código vai aqui */ }
